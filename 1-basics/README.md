# typescript-tutorial

## Basic setup

- compiling ts to js

```
tsc app.ts
```

- lite server

```
  "scripts": {
    "start": "lite-server"
  },
    "devDependencies": {
    "lite-server": "^2.5.4"
  }
```

=> serves the index.html to the browser
=> don't have to open live server every time you change
=> automatic reloading
